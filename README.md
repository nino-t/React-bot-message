## React bot message
React bot message use react and snatchbot for bot third party
<br>
![alt text](https://github.com/nino-t/React-bot-message/blob/master/demo/demo.gif)

### Installing

A step by step series of examples that tell you how to get a development env running

Say what the step will be

```
npm install
```

And to running server with port 3000

```
npm start
```

You can access in web browser with this address

```
http://localhost:3000
```

End with an example of getting some data out of the system or using it for a little demo
